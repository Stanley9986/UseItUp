# UseItUp Documentation Package

This folder contains starter documentation for the UseItUp mobile app MVP.

## Files

- `product-spec.md` — product definition, MVP scope, features, risks, and future ideas
- `use-cases.md` — core use cases for pantry tracking and recipe flow
- `data-model.md` — MVP database schema and pantry update logic
- `workflow-and-screens.md` — user workflow and Figma/mobile screen notes
- `tech-stack.md` — recommended tech stack and build phases
- `codex-instructions.md` — instructions to give Codex when starting implementation

## Recommended Next Step

Start with Figma screens using `workflow-and-screens.md`, then create an Expo React Native project and give Codex `codex-instructions.md` as implementation context.
